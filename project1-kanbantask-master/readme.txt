QuickKanbanTask:

-- QuickKanbanTask is a sleek and user-friendly Kanban-style Task Management App built with React, TailwindCSS, and @dnd-kit. 
-- It enables users to manage their tasks visually through drag-and-drop columns labeled To Do, In Progress, and Done.
-- Designed for clarity and responsiveness, this app is ideal for individuals or teams looking for a minimal yet powerful tool to manage daily tasks or project workflows.

Core Functionalities:
Priority Badges:
-- Each task can be labeled with a priority: High, Medium, or Low, displayed as color-coded badges for quick identification.

Drag-and-Drop Tasks:
-- Utilizes @dnd-kit to enable intuitive drag-and-drop functionality so users can move tasks seamlessly across different status columns.

Create/Edit/Delete Tasks:
-- Users can easily create new tasks, update existing ones via a modal popup, or delete tasks they no longer need.

Global Task State:
-- All task data is globally managed using React Context API, ensuring smooth state flow across components.

Clean and Responsive UI:
-- TailwindCSS powers a modern, mobile-responsive layout, keeping interactions fast, clean, and elegant.

Interactive UX:
–- Modal-based task input/edit with smooth transitions and icons.

Data Persistence:
-– LocalStorage setup ready to prevent data loss

Tech Stack:
=> React JS - Frontend UI library
=> Tailwind CSS  - Utility-first CSS framework        
=> @dnd-kit/core - Drag-and-drop interactions         
=> @dnd-kit/sortable - Sortable drag-n-drop lists         
=> Context API - React built-in state management    
=> Font Awesome - Icon library for UI enhancement
